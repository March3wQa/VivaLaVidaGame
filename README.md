# Viva La Vida Game
Game made in Unreal Engine 4.16
# Game is now downloadable: https://drive.google.com/open?id=0B74fy2afcLfWNThNMExXNjdwMDQ

# Controls:
W - Forward | 
S - Backward | 
A - Left | 
D - Right | 
Mouse - Camera Movement | 
Space Bar - Jump | 
C - Change Camera

# Gamepad Controls:
Left Stick - Movement | 
Right Stick - Camera Movement | 
D-Pad Down - Camera Change
